# EMI Calculator

A fast, SEO-optimized EMI (Equated Monthly Installment) calculator built with **Astro** and **Tailwind CSS v4**. Calculates monthly EMI, total interest payable, and generates full amortization schedules for home, car, and personal loans. Supports both ₹ (INR) and $ (USD).

![EMI Calculator](https://emicalculator.example.com/og-image.png)

## Features

- **Instant calculations** — results update in real-time as you type (150ms debounce)
- **Two EMI methods** — Reducing Balance (standard) and Flat Rate
- **Currency toggle** — Switch between ₹ (INR) and $ (USD) with Indian number formatting
- **Full amortization schedule** — month-by-month breakdown, first 12 visible, rest collapsible
- **Dark mode** — follows system preference, toggleable via header button
- **Mobile-first** — responsive design with 44px touch targets
- **SEO-ready** — JSON-LD schemas (FAQPage + WebApplication), Open Graph, semantic HTML
- **AdSense-ready** — placement stubs in Layout + Footer, ads.txt included
- **Static export** — zero backend, zero server-side code

## Quick Start

### Prerequisites

- **Node.js** 18 or higher
- **npm** 9 or higher

### Local Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Open in browser
# → http://localhost:4321
```

### Build for Production

```bash
# Build static site
npm run build

# Preview the build locally
npm run preview

# Output directory: dist/
```

## Deploy to Cloudflare Pages

1. Push your code to a GitHub repository
2. Go to [Cloudflare Pages](https://pages.cloudflare.com) → Create a project
3. Connect your GitHub repo
4. Set build configuration:
   - **Build command:** `npm run build`
   - **Build output directory:** `dist`
   - **Node.js version:** `18` (or higher)
5. Deploy!

## Configuration Checklist

Before deploying to production, update these placeholders:

### 1. Site URL

In `astro.config.mjs`, replace the `site` field with your actual domain:

```js
site: 'https://yourdomain.com',
```

### 2. Google Analytics (GA4)

In `src/layouts/Layout.astro`, uncomment and update the GA4 script:

```html
<!-- Replace G-XXXXXXXXXX with your Measurement ID -->
<script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
```

### 3. Google AdSense

In `src/layouts/Layout.astro`, uncomment and update the AdSense script:

```html
<!-- Replace ca-pub-XXXXXXXXXXXXXXXXX with your publisher ID -->
<script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-XXXXXXXXXXXXXXXXX"></script>
```

Additional ad placements are available in:
- `src/components/Footer.astro` — footer ad unit
- `src/pages/index.astro` — mid-page ad unit

### 4. ads.txt

In `public/ads.txt`, uncomment and replace with your publisher ID:

```
google.com, pub-XXXXXXXXXXXXXXXXX, DIRECT, f08c47fec0942fa0
```

### 5. Search Console Verification

In `src/components/SeoHead.astro`, uncomment and add your verification codes:

```html
<meta name="google-site-verification" content="YOUR_CODE">
<meta name="msvalidate.01" content="YOUR_CODE">
```

### 6. Sitemap

The sitemap is auto-generated by `@astrojs/sitemap`. It uses the `site` URL from `astro.config.mjs`. After setting your domain, the sitemap will be available at `/sitemap-index.xml`.

Update `public/robots.txt` to point to your actual sitemap URL.

### 7. Contact Email

In `src/pages/contact.astro`, replace `hello@emicalculator.example.com` with your actual email address.
